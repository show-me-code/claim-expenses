# 差旅费用报销系统

用于帮你整理住宿费和火车票发票，自动帮你整理闭环，并返回excel帮助填写。

智能整理行程发票，一键生成报销汇总。

## 功能特点

- ✅ 自动解析PDF火车票（12306电子发票）
- ✅ 自动解析PDF住宿发票（增值税电子发票）
- ✅ 智能配对行程闭环（支持多种模式）
- ✅ 自动计算费用（交通+住宿+伙食补助）
- ✅ 生成Excel报销汇总表
- ✅ 防重复计算机制
- ✅ 苹果风格Web界面

## 闭环识别模式

系统支持两种闭环识别模式：

| 模式 | 说明 | 识别逻辑 |
|------|------|----------|
| **标准闭环** | 北京 → 出差地 → 北京 | 去程从北京出发，返程回到北京 |
| **休假期间出差** | 休假地 → 出差地 → 北京 | 去程从休假地出发，返程回到北京 |

**注意：** 只有形成闭环的行程才计入费用，不闭环的票据会被跳过。

## 使用说明

### 快速启动

**方式一：双击启动脚本**
```
双击 run.py 或 start.bat
```

**方式二：命令行启动**
```bash
cd src/backend
python app.py
```

启动后浏览器会自动打开 http://localhost:5000

### 使用流程

1. **选择发票文件夹**
   - 输入包含PDF发票的根目录路径
   - 例如：`C:\Users\你的用户名\Documents\发票文件夹`

2. **设置伙食补助**
   - 默认100元/天
   - 可自定义修改

3. **点击处理**
   - "开始处理"：跳过已处理的发票（防重复计算）
   - "重新计算所有"：强制重新处理所有发票

4. **查看结果**
   - 系统显示费用汇总
   - 显示各行程明细
   - 下载Excel报表

### 防重复计算

系统会记录已处理的发票号码：
- 第一次处理：计入费用并记录发票号
- 第二次处理：自动跳过已记录的发票
- 清除记录：点击"清除记录"按钮可重置

### 文件组织

处理完成后，系统会在原文件夹下创建按日期命名的子文件夹：

```
发票文件夹/
├── 2026-01-25-2026-01-30/    # 自动创建
│   ├── [dep] 去程车票.pdf
│   ├── [ret] 返程车票.pdf
│   ├── [sup] 退票费.pdf
│   ├── 住宿发票.pdf
│   └── summary.txt            # 费用汇总
```

### Excel报表

生成的Excel包含：
- 报销汇总表（日期、行程、各项费用）
- 详细明细（车次、酒店、补助）
- 小计和总计

## API文档

### 基础接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/health` | GET | 健康检查 |
| `/api/process` | POST | 处理发票 |
| `/api/processed` | GET | 获取已处理记录 |
| `/api/clear` | POST | 清除处理记录 |
| `/api/download/<filename>` | GET | 下载Excel |

### 处理接口详情

**POST `/api/process`**

请求参数：
```json
{
    "rootFolder": "发票文件夹路径",
    "dailyMealRate": 100,      // 伙食补助标准
    "skipProcessed": true      // 是否跳过已处理
}
```

响应：
```json
{
    "success": true,
    "trips": [...],            // 行程列表
    "expenses": {
        "grand_total": 3603.5, // 总金额
        "total_tickets": 997,  // 交通费
        "total_hotels": 1900,  // 住宿费
        "total_meals": 600     // 伙食费
    },
    "processed_count": {
        "new_tickets": 3,      // 新处理车票数
        "new_invoices": 1      // 新处理发票数
    }
}
```

## 常见问题

### Q: 为什么金额是0？

A: 可能原因：
1. 所有发票已被处理过（点击"清除记录"重新计算）
2. 票据没有形成闭环（没有返程票）
3. 文件夹路径不正确

### Q: 如何处理退票费？

A: 退票费发票会被自动匹配到对应行程，计入交通费用。

### Q: 支持哪些发票格式？

A: 目前支持：
- 12306电子发票（铁路电子客票）
- 增值税电子发票（普通发票）- 住宿服务类

在 `trip_matcher.py` 中修改 `match_trips()` 函数。
